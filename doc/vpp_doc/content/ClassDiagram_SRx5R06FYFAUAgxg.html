Model Elements
Name Description
  MGrid : Class

Grid is a Pure Fabrication class to manage forest tokens. It creates, knows and manages them.

It is responsible for the forest tokens creations, to face up a single token to get it value.

It has a reference to a strategy for choosing the next forest token and for shuffling the tokens.

  CollectionsShuffleStrategy : Class

It is a class responsible to shuffle using the Collection.shuffle() method.

  IShuffleStrategy : Class

It is a Strategy Interface to shuffle.

  RandomOrgAdapter : Class

It is a class responsible to shuffle using the Random.org external service.

  NoShuffleStrategy : Class

It is a class responsible to no shuffle.

  ShufflingRedirectionProxy : Class

It is responsible to make a redirection proxy for the shuffing.
