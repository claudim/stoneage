Use Case - Istruisci Player link

Properties
Name Value
Description

Un giocatore vuole conoscere le regole del gioco prima di iniziare a giocare. Per questo richiede al sistema un tutorial. Il sistema richiede al giocatore di scegliere la modalità di gioco per cui vuole essere istruito. Una volta che il giocatore ha effettuato la scelta della modalità di gioco, il sistema mostra al giocatore le regole del gioco attraverso un tutorial. Al termine del tour il giocatore può scegliere di giocare o ripeterlo.
