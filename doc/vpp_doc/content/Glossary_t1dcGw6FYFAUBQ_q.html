Description

Un elemento del game board del gioco. L'insieme delle caselle costituisce il percorso su cui il marker di un giocatore può muoversi.

Alcune caselle appartengono a più di una modalitàdel gioco.

Le caselle sono: Mercato, Cantiere, Cane, Sorpresa, Prateria, Foresta, Artigiano, Cava, Campo di Battaglia, Fiume.
