Use Case - Consiglia Mossa link

Properties
Name Value
Description

Il giocatore vuole essere consigliato sulla prossima mossa da eseguire:

Durante lo svolgimento della partita il giocatore non sa quale mossa effettuare perciò richiede al sistema di indicargli una possibile mossa successiva. Di conseguenza il sistema deve individuare la prossima mossa sulla base dello stato del gioco e mostrarla all’utente. A questo punto l’utente conosce la prossima possa che può effettuare. La decisione ora spetta a lui.
