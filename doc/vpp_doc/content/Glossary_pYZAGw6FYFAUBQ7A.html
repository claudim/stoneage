Description

Un elemento del gioco utilizzato per la costruzione di una capanna, per lo scambio nel mercato o come ricompensa per la vittoria di una battaglia.

Le risorse possono essere distinte in:

  1. Bacca
  2. Dente
  3. Freccia
  4. Pesce
  5. Anfora
