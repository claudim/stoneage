Description

Contratto CO1: playStoneAge IT3

 

Operazione:                playStoneAge(mode:String, numPlayers: Integer, markerName: String)

Riferimenti:                casi d’uso: Avvia Partita

Pre-condizioni:           Il giocatore sta avviando la partita in modalità rossa(verde)

Post-condizioni:        

                                 -    è stata creata un’istanza sag di StoneAgeGame

-       sono state create 16(14) istanze di TokenForest

-       per ciascuna istanza di TokenForest, l’attributo state è diventato faceDown

-       ciascuna istanza di TokenForest è stata associata con sag

-       è stata creata un’istanza board di Board

-       sag è stata associata con board

-       sono state create 10(8) istanze di Square: un'istanza di buildingSiteSquare, 6 istanze di resourceSquare, 1(0) istanze di surpriseSquare, 3(1) istanze regularSquare

-       ciascuna istanza di Square è stata associata con board

-       ciascuna istanza di Square è stata associata ad un'istanza di Square

- sono state create 5 istanze di tipo resource per ogni resourceSquare e 2 istanze per la resourceSquare cane

- sono state associate le istanze di resource alle rispettive square

- sono state create 15 istanze di hutToken

- sono state associate le istanze di hutToken alla buildingSiteSquare

- sono state create 4(0) istanze di surpriseToken

- sono state associate le istanze di surpriseToken create alla surpriseSquare

-       sono state create numPlayers istanze di Player

-       ogni istanza di Player è stata associata a sag

-       sono state create numPlayers istanze di Marker 

-       un’istanza di Marker è stata associata ad una e una sola istanza di Player

-       tutte le istanze di Marker sono state posizionate su (associate con) la casella di partenza (una delle istanze di Square)

-       sono state create numPlayers istanze di Settlement 

-       un’istanza di Settlement è stata associata ad una e una sola istanza di Player
